## EscaleX (EX)
 
O EscaleX é uma aplicação criada para atender o controle do pessoal da escala de serviço do DCMUN.
 
 
## Tecnologias 
 
Aqui estão as tecnologias utilizadas nesse projeto.
 
* Reactjs
* Nextjs
* Styled Components
* Google Spreadsheets como Database (lib: Tabletop)
 
 
## Serviços utilizados
 
* Github
* Vercel
